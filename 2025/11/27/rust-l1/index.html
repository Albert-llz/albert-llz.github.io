Rust 冒泡排序示例

背景

冒泡排序（Bubble Sort）是一种稳定、基于比较的排序算法。它通过多轮遍历，将相邻的逆序元素交换位置，让较大的元素逐步“冒泡”至序列末尾；若某一轮未发生交换，则说明序列已经有序，可提前结束。

特性 描述
最坏/平均时间复杂度 O(n²)
最好时间复杂度 O(n)（序列已排序且配合提前退出优化）
空间复杂度 O(1) 原地排序
稳定性 稳定

代码说明

src/bubble_sort.rs 中实现了一个泛型的 bubble_sort 函数，只要元素实现了 Ord trait 即可使用。文件还包含：

  • main：演示对整数向量的排序。
  • tests 模块：提供整数与字符串两组单元测试，验证算法在不同数据类型上的表现。

Rust 二分插入排序示例

背景

二分插入排序（Binary Insertion Sort）是插入排序的优化版：每次插入时先用二分查找定位插入位置，再把元素移动到正确位置，从而减少比较次数。虽然整体时间复杂度仍为 O(n²)，但在比较成本高的场景下更具优势。

特性 描述
最坏/平均时间复杂度 O(n²)
最好时间复杂度 O(n)（列表已排序）
空间复杂度 O(1) 原地排序
稳定性 稳定

Rust 递归排序（归并排序）示例

背景

归并排序（Merge Sort）是经典的分治与递归排序算法：将序列拆分成两半分别排序，再将两个已排序子序列合并。虽然需要额外内存，但提供稳定、可预测的 O(n log n) 性能，非常适合大规模数据或需要稳定性的场景。

特性 描述
时间复杂度 O(n log n)（最坏/平均/最好）
空间复杂度 O(n)（合并阶段的辅助空间）
稳定性 稳定
关键思想 分治 + 递归
